Check it out here: [money043.github.io](https://money043.github.io)

## 🛠 Tech Stack
This site was built with performance, animation, and dynamic user interfaces in mind.
- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Animations**: GSAP (ScrollTrigger) & Framer Motion
- **Video Handling**: HLS.js
- **Icons**: Lucide React
- **Deployment**: GitHub Actions -> GitHub Pages

- **Streamify**: Analytics & Media streaming.
- **PrintQR**: QR-based print management solution.
- **Other Notebooks**: Machine Learning, LLM-DevOps Assistant, and NLP Resume Parsers.
